Product Description
TonyPi intelligent visual humanoid robot uses Raspberry Pi as the central controller. OpenCV is the image processing library, which uses Python language programming. The robot is equipped with a 2DOF high-definition camera to realise Al many games. It equipped with high-voltage bus servo and voltage display. With built-in action groups, TonyPi can complete many challenging motion like push-ups.
Feature
  • The robot kit includes the Raspberry Pi 4B 4GB controller board.
  • Large capacity battery, long working duration
  • Robot battery voltage display
  • Installed HD camera catches real-time images
  • Provide PC software source code and Python source code
  • Support PC software, phone App and VNC remote desktop control
Vision Auto Shooting
Perform image processing through OpenCV to obtain the ball’s position, and then use PID algorithm to track and kick it automatically.
Color Recognition
Use OpenCV to convert the image into LAB color space, then obtain the recognized color. TonyPi nods when recognizes red and shakes for the green and blue.
Tag Recognition (AprilTag)
TonyPi performs actions according to the recognized tags. In addition to the provided tags, you’re encouraged to program your own tags and actions.
Facial Recognition
Use OpenCV to load the trained face detection model. TonyPi will operate the greeting action when recognizes a face.
Color Tracking
Processing images through OpenCV to receive positions of red, green and blue, TonyPi tracks the target color in real time.
Line Following
TonyPi can recognize black, white and red line through OpenCV. Calculate the line’s position and then adjust the route to follow.

Control Method
Provide PC software and APP for TonyPi's control. In the “WonderPi” App, you can connect it to the wireless LAN instead of searching IP in router to set the MAC, which is easier!
High Voltage Bus Servo
11.1V high voltage bus servo reduces current by more than 60%, which improves the robot’s service time.
2DOF Wide-angle Camera
Loaded with 120° wide angle HD camera, TonyPi can look uo and around for wider vision. Support manual focus adjustment.
Powerful Control System
  • Independent power switch for stable power supply
  • channel PWM servo port with avoiding current protection
  • Loaded 6-axis sensor IC helps data collection
  • GPIO pins and 4 IIC ports for users’ extension
  • 4PIN ports are compatible with Hiwonder sensors
